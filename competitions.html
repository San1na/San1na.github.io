🏆 Guild Competitions

Guild Competitions are server-wide events where guilds compete in specific farming categories to earn massive rewards.


📊 Event Categories (Tags)

When a competition starts, the server tracks one of the following hidden tags:

  • carrots_harvested: Tracks the total number of carrots harvested by all guild members.
  • golden_carrots_harvested: Tracks only Golden Carrots (mutation variants).
  • plant_weight_harvested: Tracks the total physical weight (in grams/kg) of all plants harvested.
🤝 Contribution System

Every time you harvest a plant, the client sends a Report(player, tag, amount) network packet. The server validates the numeric amount against your current plant's theoretical maximum weight to prevent exploiting.

🌐 Sync & Leaderboards

Competition data is dynamically formatted before being shown on the leaderboard. For example, plant_weight_harvested is automatically converted from raw integer values into readable weights like "15.4 kg" or "1.2 tons".

The leaderboard is synced periodically via the GuildTickUpdate and GuildGetLeaderboard network endpoints.
