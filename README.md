# 🚜 Grow A Garden 2 Wiki

**🌐 Live Website:** [https://san1na.github.io/](https://san1na.github.io/)

Welcome to the ultimate datamined Wiki for **Grow A Game 2**. 

This project is a static HTML/CSS wiki generated by directly analyzing the game's `SharedModules` and `SharedData` source code. It reveals the hidden math, exact multipliers, and exact item stats that the game doesn't explicitly tell you.

## 🌟 Features

* **🌱 Seeds & Plants Database:** Exact growth times, base prices, and the hidden math behind plant weight multipliers and mutation penalties.
* **🛠️ Tools & Gears Database:** Complete stats for all sprinklers, watering cans, and utilities. **Proof that sprinklers do NOT stack!**
* **🐾 Pets Database:** Drop chances and the exact scaling multipliers for Huge and Rainbow pets.
* **🏆 Guild Competitions:** An explanation of how the server tracks and converts weight and harvest amounts for leaderboard syncing.
* **🌤️ Hidden Mechanics:** Discover how weather events modify your farm and the exact mathematical curve for offline/overtime growth limits.

## 🚀 How to use this Wiki locally

This wiki requires zero server setup. It is built entirely with plain HTML and CSS.

1. Clone or download this repository to your computer.
2. Double-click on `index.html` to open it in your favorite web browser.
3. Browse the tabs to explore the stats!

## ⚠️ Datamining Notice

Because the source code was dumped from a Roblox executor, the raw Roblox asset IDs for images are trapped inside instance `StringValues` that were not saved. To keep the wiki functional and fast, we use standard emojis instead of the official game icons.
